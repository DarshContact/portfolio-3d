# 🚀 Darsh - 3D Portfolio Website

A stunning, creative 3D portfolio website built with modern web technologies.

## ✨ Features

- **3D Hero Section**: Interactive Three.js icosahedron with particle system
- **Custom Cursor**: Smooth follower cursor with blend modes
- **Particle Background**: Animated particle network with connections
- **3D Card Effects**: Flip cards and tilt effects on hover
- **Typing Animation**: Dynamic text rotation
- **Smooth Scrolling**: Parallax effects and scroll animations
- **Responsive Design**: Mobile-friendly with hamburger menu
- **Contact Form**: Saves submissions to CSV file
- **Dark Theme**: Neon cyan and purple accents

## 🛠️ Technologies Used

- HTML5
- CSS3 (3D transforms, animations)
- Vanilla JavaScript
- Three.js (3D graphics)
- Font Awesome (icons)
- Google Fonts (Poppins, Orbitron)

## 🚀 Quick Start

### Option 1: Local Development

1. Clone the repository:
```bash
git clone https://github.com/todarsh002/portfolio-3d.git
cd portfolio-3d
```

2. Open `index.html` in your browser (no build required!)

### Option 2: GitHub Pages (Live)

Visit: **https://todarsh002.github.io/portfolio-3d/**

## 🎨 Customization

### Change Colors

Edit `styles.css` root variables:

```css
:root {
    --primary-color: #00f7ff;      /* Cyan */
    --secondary-color: #ff00ff;    /* Magenta */
    --accent-color: #7b2cbf;       /* Purple */
}
```

### Update Personal Info

Edit `index.html`:
- Name, title, bio
- Social media links
- Projects and experience
- Contact information

### Modify 3D Elements

Edit `script.js`:
- Change icosahedron size/color
- Adjust particle count
- Modify animation speeds
